## drip.py / Arduino IV Drip Monitoring Prototype.

This project explores a low-cost hardware approach to monitoring the drip rate of an IV infusion.

An optical sensor connected to an Arduino detects individual drops passing through the drip chamber. Each detected drop is recorded and processed to estimate the drip rate over time.

The collected sensor data is analysed using Python and visualised in MATLAB to examine signal behaviour, measurement stability, and potential filtering approaches.

The project is intended as a small exploration of how simple embedded systems and data analysis can be applied to real-world clinical monitoring problems.

**Tech stack**
- Arduino
- Python
- MATLAB
