# Media

This folder contains visual material for the infusion drip monitoring prototype.

## Structure

- `demo_video/`  
  contains the final demonstration video of the prototype

- `photos/`  
  contains photos of the hardware setup, build process, and final prototype

- `sketches_and_schematics`  
  contains early handwritten notes and concept sketches, as well as the
  final breadboard schematic




## Note
The media files are included to support the written documentation and show both the final result and the development process of the project.
