# Documentation

This folder contains the written documentation for the infusion drip monitoring prototype.
The files are organised by topic and follow the development of the project from the first idea to the final reflection.

## Contents

- `01_problem_and_motivation.md`  
  explains the general problem, the healthcare context, and why I chose this topic

- `02_project_goals.md`  
  describes the main goals of the project and what the prototype was supposed to do

- `03_materials_and_sources.md`  
  lists the components, materials, and software used in the project

- `04_hardware_setup.md`  
  describes how the physical prototype was built and arranged

- `05_challenges_and_iterations.md`  
  documents the main problems during development and how the project changed over time

- `06_circuit_and_pinout.md`  
  shows the final circuit logic and pin assignment

- `07_code_development.md`  
  explains how the Arduino code developed and what the final version does

- `08_design_decisions.md`  
  reflects on important design choices made during the project

- `09_kalman_filter_brief.md`  
  briefly explains why filtering is relevant and how the project connects to later MATLAB analysis

- `10_reflection_and_future_improvements.md`  
  gives a final reflection on the project and possible next steps

## Note
This documentation is meant to show not only the final result, but also the development process behind the prototype. Because of that, it includes both technical explanations and reflections on testing, limitations, and design decisions.
